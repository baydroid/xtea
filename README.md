### A simple file encryption and decryption tool using the XTEA implementation from mbed TLS (https://tls.mbed.org).

Install Zig from https://ziglang.org/download/

Clone the this xtea repository, cd into it, and issue the command

    zig build

Run xtea

    zig-out/bin/xtea

and enter help at the xtea> prompt.

    xtea> help
